# Архитектура проекта

Проект **UploadBot** разделен на несколько логических модулей, каждый из которых отвечает за свою часть обработки видео. Такое разделение облегчает поддержку и добавление новых функций.

## Основной модуль

- **`bot.py`**: Точка входа в приложение. Инициализирует Telegram-бота, регистрирует обработчики (handlers) и запускает основную асинхронную петлю.

## Модули-обработчики (handlers/)

Для избежания перегруженности основного файла `bot.py`, обработчики Telegram-событий вынесены в отдельную директорию:
- **`commands.py`**: Обработчики базовых команд (`/start`, `/settings`, `/preview`, и т.д.).
- **`callbacks.py`**: Обработка нажатий на inline-кнопки (callback query).
- **`message_handlers.py`**: Обработка текстовых сообщений (например, ссылок на YouTube).
- **`processing.py`**: Содержит основной пайплайн (очередь) начальной обработки ссылок и запуска задач.
- **`mailru_handlers.py`**: Функции для взаимодействия с Mail.ru Cloud.
- **`helpers.py`**: Вспомогательные классы (например, фейковые объекты сообщений для имитации интерфейса).

## Модули обработки данных

- **`download.py`**: Скачивание видео с YouTube с помощью `yt-dlp`.
- **`transcribe.py`**: Транскрибация аудио в текст с использованием модели OpenAI Whisper. Поддерживает кэширование загруженной модели.
- **`scenes.py`**: Нарезка длинного видео на сцены с помощью библиотеки `PySceneDetect`.
- **`autoedit.py`**: Удаление тишины и пауз в аудиодорожке с помощью `auto-editor`.
- **`vertical.py`**: Преобразование видео в вертикальный формат (9:16). Включает логику `facecam_top_split`, при которой верхняя часть экрана фокусируется на камере (поиск лица `yolo_window_v1`), а нижняя - на контенте.
- **`subtitles.py`**: Генерация и наложение стилизованных ASS/SSA субтитров.
- **`tagging.py`**: AI-генерация хештегов для социальных сетей.
- **`llm.py`**: Модуль для суммаризации текста и генерации описаний (используются платформы OpenAI или Ollama).
- **`moments.py`**: AI-анализ транскрипции для выделения наиболее интересных (вирусных) моментов (лучшие моменты видео).

## Инфраструктурные модули

- **`security.py`**: Шифрование (через `Fernet`) и дешифрование конфиденциальных данных, валидация URL-адресов.
- **`cache.py`**: Система логирования и работы с временными файлами (кэшем) приложения.
- **`settings.py`**: Управление пользовательскими настройками в форме JSON с сохранением состояния.
- **`utils.py`**: Вспомогательные функции, общие для всех модулей проекта.
- **`mailru.py`**: API для интеграции с Mail.ru Cloud (сохранение результатов в облако).
