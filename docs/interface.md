# Интерфейс и команды

Взаимодействие с UploadBot происходит через Telegram-интерфейс, который включает текстовые команды, отправку видео/ссылок и интерактивные (inline) кнопки-меню.

## Основные команды

Ниже представлен список доступных команд:

| Команда | Описание |
|---------|----------|
| `/start` | Инициализация работы с ботом, приветственное сообщение и создание базовых настроек пользователя. |
| `/settings` | Вызов интерактивного меню настроек обработки видео (см. раздел "Меню настроек"). |
| `/preview` | Создание короткого превью с текущими настройками кадрирования и наложения. |
| `/moments` | Запуск алгоритма поиска лучших (вирусных) моментов из уже загруженных видео. |
| `/llm` | Проверка статуса LLM (OpenAI или Ollama) и выбор провайдера для генерации описаний. |
| `/gpu` | Проверка доступности GPU (CUDA) для ускорения транскрибации Whisper и других AI-задач. |
| `/cache` | Проверка текущего состояния кэша (временных файлов) и очистка места. |
| `/mailru` | Интеграция с облачным хранилищем Mail.ru Cloud для авто-бэкапа готовых видео. |

## Отправка видео

Основной сценарий использования бота — отправка ссылки на YouTube или загрузка видеофайла.
1. Пользователь отправляет ссылку на YouTube (например, `https://youtu.be/...`).
2. Бот проверяет валидность ссылки.
3. Видео ставится в очередь, и пользователь получает статусное сообщение, которое обновляется по мере прохождения этапов (скачивание 🟢, транскрибация 🟡, нарезка 🔴 и т.д.).

## Меню настроек (`/settings`)

Команда `/settings` открывает удобное меню на основе inline-кнопок, где можно настроить процесс под конкретные задачи. Каждое изменение сохраняется индивидуально для пользователя.

Основные параметры, доступные в настройках:
- **Разрешение и качество**: Настройка исходного качества при скачивании `yt-dlp`.
- **Режим кадрирования (Crop Mode)**:
  - `center` — фокус на центре кадра.
  - `facecam_top_split` — алгоритм разделения экрана `[Камера сверху / Контент снизу]`. (Поиск камеры через `yolo_window_v1`).
- **Язык транскрибации**: Выбор языка для Whisper (или автоопределение).
- **Стиль субтитров**: Включение/выключение вшитых субтитров и выбор их дизайна.
- **Интеграции**: Включение отправки результатов в подключенные облачные сервисы (например, Mail.ru).

## Возврат результатов

По завершении обработки бот присылает пользователю:
1. Итоговое видео (в формате 9:16).
2. Сгенерированное название и краткое описание (LLM).
3. Список актуальных хештегов (`tagging.py`).
4. Если активирован экспорт — ссылку на облачный диск.
