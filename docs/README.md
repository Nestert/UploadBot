# Документация UploadBot

Добро пожаловать в документацию проекта UploadBot. Здесь собрана подробная информация об архитектуре проекта, алгоритмах работы и инструкциях по развертыванию.

## Оглавление

1. [Архитектура проекта](architecture.md) - Описание модулей и их взаимодействия.
2. [Развертывание и настройка](setup.md) - Инструкции по запуску с помощью Docker и локальной установке.
3. [Обработка видео](video_processing.md) - Описание пайплайна: скачивание, транскрибация, нарезка и наложение субтитров.
4. [Интерфейс и команды](interface.md) - Описание команд Telegram-бота и меню настроек.

В дальнейшем этот раздел будет пополняться новыми документами по мере развития проекта.
